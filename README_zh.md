## 使用

### 前置条件

- Git
- 一个工作空间干净并且在 `gh-pages` 分支具有上传权限的存储库

1. 进入你的存储库路径
```shell
cd your_repository_path
# 例如：
# cd git-green
```
2. 运行命令
```shell
green
```
